# Encouragement by Empowerment (EbE)

Encouragement by Empowerment is a modern, community-driven web application dedicated to uplifting individuals and families through mentorship, resource allocation, and sustainable support networks.

## 🌟 Mission
Our mission is to break cycles of struggle and build a stronger, more resilient community by providing personalized mentorship and direct access to essential resources.

## 🚀 Key Features

- **Responsive Design:** Optimized for mobile, tablet, and desktop experiences.
- **Impact Tracking:** Real-time visualization of community impact statistics.
- **Secure Donations:** A fully functional donation system with credit card validation and secure processing simulation.
- **Program Management:** Detailed information about mentorship, resource allocation, and community networks.
- **Interactive UI:** Smooth animations and transitions powered by `motion/react`.
- **Modern Tech Stack:** Built with performance and maintainability in mind.

## 🛠️ Tech Stack

- **Frontend:** React 18+, TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Motion (framer-motion)
- **Icons:** Lucide React
- **Build Tool:** Vite
- **Routing:** React Router DOM

## 📂 Project Structure

- `/src/components`: Reusable UI components (Navbar, Footer, Stats, etc.)
- `/src/pages`: Main application pages (Home, About, Donate, etc.)
- `/src/index.css`: Global styles and Tailwind configuration.
- `/src/App.tsx`: Main routing and application layout.

## 📦 Getting Started

1. **Clone the repository**
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Run the development server:**
   ```bash
   npm run dev
   ```
4. **Build for production:**
   ```bash
   npm run build
   ```
